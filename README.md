# LEARNING_PLAYWRIGHT

Collection of Playwright learning examples and JavaScript tutorials.

Getting started

- Prerequisites: Node.js (16+), npm
- Install dependencies (if you add Playwright tests):

```bash
npm install --save-dev @playwright/test
npx playwright install
```

- Run Playwright tests:

```bash
npx playwright test
```

Project layout

- `Chapter_01_Basics/` — basic JS examples
- `Chapter_02_Core_JS/` — core JS concepts
- `chapter_03_Identifier_Literal_Operators_Statement/` — identifiers & literals
- `chapter_04_Operators/` — operators
- `chapter_05_statements/` — statements
- `chapter_06_Switch_statements/` — switch examples
- `chapter_07_loops/` — loops
- `Programs_Tasks_Practice/` — practice programs
